# redis-node
 A CRUD application using
 - Node, Express, Redis, MySQL
 - Vue

 ### BACKEND
 - NODE
 - EXPRESS
 - REDIS - Cache Management

 ### FRONTEND
 - Vue
 - Vuetify - UI Material Framework
 - Vuex - State Management
